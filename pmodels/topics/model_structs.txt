Specifications of models
============================

This document specifies the data structures that represent several
topic models

1. Solution to Topic Latent Dirichlet Allocation

(2.a) For variational E-M algorithm, the solution s is a struct with 
the following fields:

Let n be the number of documents in the corpus.

s.K:            The number of topics
s.V:            The size of vocabulary
s.alpha:        The Dirichlet prior param (over topic distribution theta)
s.Beta:         The word distribution for topics [V x K]
s.Gamma:        The per-doc posterior Dirichlet distribution parameter of 
                topic distribution [K x n]
